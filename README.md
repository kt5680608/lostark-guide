### 소개

> 로스트아크 게임을 처음 시작하는 유저들을 위한 가이드 웹사이트입니다. 게임 진행에 있어 꼭 필요한 부분을 설명하는 가이드 및 용어 설명, 여러 캐릭터를 육성하고 있어 매일 해야하는 숙제 체크가 어렵고 헷갈리는 유저들을 위한 숙제 검사기를 제작하였습니다.<br/>
> forkoko 바로가기 <a>http://forkoko.site/</a>
